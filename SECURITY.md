# Security Policy

If you discover a security vulnerability in Mando, please report it responsibly.

**Preferred:** Use [GitHub Security Advisories](https://github.com/tribedotrun/mando/security/advisories/new) to report the issue privately.

**Alternative:** Email gm@aburra.xyz.

We aim to acknowledge reports within 48 hours.

Please do not open public GitHub issues for security vulnerabilities.
